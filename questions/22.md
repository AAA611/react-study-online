# React Output Question 22: Lazy initializer in useState runs only once

## Basic Info
- Category: Hooks / useState
- Difficulty: Junior
- Tags: lazy initializer, initial render, useState
- Type: Read React code and write the output

## Description
点击按钮后，初始化函数还会不会再次执行？

## Code
```jsx
function App() {
  const [count, setCount] = React.useState(() => {
    console.log("init");
    return 0;
  });

  console.log("render", count);
  return <button onClick={() => setCount(c => c + 1)}>{count}</button>;
}

render(<App />);
```

## Answer
Output: init, render 0, then after one click render 1.

## Key Points
- lazy initialization happens only on mount

## Detailed Explanation
`useState` 的函数初始化器只会在首次挂载时执行一次，用来计算初始 state。后面的重新渲染都会直接复用 React 保存的 state，因此再次点击时不会重新打印 `init`。
