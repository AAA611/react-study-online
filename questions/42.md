# React 输出题 42：defaultValue 只在首次挂载时生效

## 基本信息
- 分类：表单 / 非受控输入
- 难度：中级
- 标签：defaultValue, input, 非受控
- 题型：阅读 React 代码并写出输出

## 题目描述
effect 把 `text` 从 `hello` 改成 `world` 之后，input 中显示什么？

## 代码
```jsx
function App() {
  const [text, setText] = React.useState("hello");

  React.useEffect(() => {
    setText("world");
  }, []);

  return <input defaultValue={text} />;
}

render(<App />);
```

## 参考答案
输入框里仍然显示 `hello`。

## 考察点
- `defaultValue` 只会设置非受控输入框的初始值

## 详细讲解
`defaultValue` 只在 input 初次挂载时生效，用来设置非受控输入框的初始值。之后 React state 即使更新成 `world`，也不会自动覆盖 input 当前显示的内容。
