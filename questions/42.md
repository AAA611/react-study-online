# React Output Question 42: defaultValue only applies on the first mount

## Basic Info
- Category: Forms / uncontrolled input
- Difficulty: Intermediate
- Tags: defaultValue, input, uncontrolled
- Type: Read React code and write the output

## Description
effect 把 `text` 从 `hello` 改成 `world` 之后，input 中显示什么？

## Code
```jsx
function App() {
  const [text, setText] = React.useState("hello");

  React.useEffect(() => {
    setText("world");
  }, []);

  return <input defaultValue={text} />;
}

render(<App />);
```

## Answer
The input still shows hello.

## Key Points
- defaultValue sets only the initial value of an uncontrolled input

## Detailed Explanation
`defaultValue` 只在 input 初次挂载时生效，用来设置非受控输入框的初始值。之后 React state 即使更新成 `world`，也不会自动覆盖 input 当前显示的内容。
