# React Output Question 39: false, null, and undefined are ignored in JSX

## Basic Info
- Category: JSX basics
- Difficulty: Junior
- Tags: JSX, nullish rendering, children
- Type: Read React code and write the output

## Description
这个 `div` 里最终真正渲染出来的是什么？

## Code
```jsx
function App() {
  return <div>{false}{null}{undefined}{"React"}</div>;
}

render(<App />);
```

## Answer
Only React appears.

## Key Points
- false null and undefined are ignored as children

## Detailed Explanation
React 在渲染 children 时会直接跳过 `false`、`null` 和 `undefined` 这些“空节点”。因此前面三个值都不会显示出来，最终只有字符串 `React` 会出现在页面中。
