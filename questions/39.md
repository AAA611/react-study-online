# React 输出题 39：false、null 和 undefined 在 JSX 中会被忽略

## 基本信息
- 分类：JSX / 基础
- 难度：初级
- 标签：JSX, 空值渲染, children
- 题型：阅读 React 代码并写出输出

## 题目描述
这个 `div` 里最终真正渲染出来的是什么？

## 代码
```jsx
function App() {
  return <div>{false}{null}{undefined}{"React"}</div>;
}

render(<App />);
```

## 参考答案
最终只会显示 `React`。

## 考察点
- `false`、`null`、`undefined` 作为 children 时会被忽略

## 详细讲解
React 在渲染 children 时会直接跳过 `false`、`null` 和 `undefined` 这些“空节点”。因此前面三个值都不会显示出来，最终只有字符串 `React` 会出现在页面中。
