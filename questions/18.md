# React Output Question 18: useLayoutEffect runs before useEffect

## Basic Info
- Category: Hooks / timing
- Difficulty: Intermediate
- Tags: useLayoutEffect, useEffect, timing
- Type: Read React code and write the output

## Description
请写出完整的输出顺序。

## Code
```jsx
function App() {
  console.log("render");

  React.useLayoutEffect(() => {
    console.log("layout");
  }, []);

  React.useEffect(() => {
    console.log("effect");
  }, []);

  return null;
}

render(<App />);
```

## Answer
Output: render -> layout -> effect.

## Key Points
- layout effects run earlier than normal effects
- render happens before either effect

## Detailed Explanation
组件函数先执行，所以 `render` 最先输出。提交 DOM 后，`useLayoutEffect` 会先同步执行，因此输出 `layout`。普通的 `useEffect` 会在更晚的时机执行，所以最后才是 `effect`。
