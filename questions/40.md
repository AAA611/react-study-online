# React Output Question 40: An array of strings renders in order

## Basic Info
- Category: JSX basics
- Difficulty: Junior
- Tags: array children, rendering, JSX
- Type: Read React code and write the output

## Description
这个 `div` 最终会显示什么文本？

## Code
```jsx
function App() {
  return <div>{["R", "e", "a", "c", "t"]}</div>;
}

render(<App />);
```

## Answer
The page shows React.

## Key Points
- arrays can be valid React children

## Detailed Explanation
React 支持直接渲染数组形式的 children。这里数组中的每一项都是字符串，所以 React 会按顺序把它们拼接显示出来，最终页面上看到的就是 `React`。
