# React 输出题 40：字符串数组会按顺序渲染

## 基本信息
- 分类：JSX / 基础
- 难度：初级
- 标签：数组子节点, 渲染, JSX
- 题型：阅读 React 代码并写出输出

## 题目描述
这个 `div` 最终会显示什么文本？

## 代码
```jsx
function App() {
  return <div>{["R", "e", "a", "c", "t"]}</div>;
}

render(<App />);
```

## 参考答案
页面最终显示的是 `React`。

## 考察点
- 数组本身也可以作为合法的 React children

## 详细讲解
React 支持直接渲染数组形式的 children。这里数组中的每一项都是字符串，所以 React 会按顺序把它们拼接显示出来，最终页面上看到的就是 `React`。
