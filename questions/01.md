# React Output Question 01: useState initial value only works on first render

## Basic Info
- Category: Hooks / useState
- Difficulty: Junior
- Tags: useState, initial state, rerender
- Type: Read React code and write the output

## Description
阅读下面的代码，写出组件首次挂载完成后控制台的输出顺序。

## Code
```jsx
function App() {
  const [count, setCount] = React.useState(0);
  console.log("render", count);

  React.useEffect(() => {
    setCount(1);
  }, []);

  return null;
}

render(<App />);
```

## Answer
Output: render 0, then render 1.

## Key Points
- useState initial value is used only on first render
- useEffect runs after commit
- setState triggers a new render

## Detailed Explanation
第一次渲染时，`count` 的初始值是 `0`，所以先输出 `render 0`。组件完成首次提交后，`useEffect` 执行并调用 `setCount(1)`，React 会安排一次新的渲染，因此随后输出 `render 1`。因为依赖数组为空，所以这个 effect 只会在首次挂载后执行一次。
