# React Output Question 27: Context consumers rerender when provider value changes

## Basic Info
- Category: Context
- Difficulty: Intermediate
- Tags: context, provider, consumer
- Type: Read React code and write the output

## Description
点击按钮前后，子组件分别会输出什么？

## Code
```jsx
const CountContext = React.createContext(0);

function Child() {
  const value = React.useContext(CountContext);
  console.log("child", value);
  return null;
}

function App() {
  const [count, setCount] = React.useState(0);

  return (
    <CountContext.Provider value={count}>
      <button onClick={() => setCount(c => c + 1)}>{count}</button>
      <Child />
    </CountContext.Provider>
  );
}

render(<App />);
```

## Answer
Output: child 0, then after one click child 1.

## Key Points
- consumers subscribe to provider value changes

## Detailed Explanation
`Child` 通过 `useContext` 订阅了 `CountContext`。当 Provider 的 `value` 从 `0` 变成 `1` 时，React 会通知消费者更新，因此子组件会重新渲染并输出 `child 1`。
