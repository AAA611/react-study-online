# React 输出题 41：Fragment 不会额外生成 DOM 节点

## 基本信息
- 分类：JSX / Fragment
- 难度：初级
- 标签：Fragment, DOM 结构, 渲染
- 题型：阅读 React 代码并写出输出

## 题目描述
下面代码里，effect 会打印多少个 `span` 元素？

## 代码
```jsx
function App() {
  React.useEffect(() => {
    console.log(document.querySelectorAll("span").length);
  }, []);

  return (
    <>
      <span>A</span>
      <span>B</span>
    </>
  );
}

render(<App />);
```

## 参考答案
最终打印的是 `2`。

## 考察点
- Fragment 本身不会变成真实的 DOM 节点

## 详细讲解
`Fragment` 只是一个语法层面的包裹容器，不会渲染成真实 DOM 元素。最终页面中真正存在的只有两个 `span`，所以打印结果是 `2`。
