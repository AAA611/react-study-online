# React Output Question 41: Fragment does not create an extra DOM node

## Basic Info
- Category: JSX / Fragment
- Difficulty: Junior
- Tags: Fragment, DOM structure, render
- Type: Read React code and write the output

## Description
下面代码里，effect 会打印多少个 `span` 元素？

## Code
```jsx
function App() {
  React.useEffect(() => {
    console.log(document.querySelectorAll("span").length);
  }, []);

  return (
    <>
      <span>A</span>
      <span>B</span>
    </>
  );
}

render(<App />);
```

## Answer
It prints 2.

## Key Points
- Fragment itself does not become a real DOM node

## Detailed Explanation
`Fragment` 只是一个语法层面的包裹容器，不会渲染成真实 DOM 元素。最终页面中真正存在的只有两个 `span`，所以打印结果是 `2`。
