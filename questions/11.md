# React 输出题 11：React.memo 遇到稳定的基础类型 props

## 基本信息
- 分类：性能 / React.memo
- 难度：中级
- 标签：React.memo, props, 重渲染
- 题型：阅读 React 代码并写出输出

## 题目描述
点击按钮一次后，`MemoChild` 会不会重新渲染？

## 代码
```jsx
const MemoChild = React.memo(function Child({ label }) {
  console.log("child render", label);
  return null;
});

function App() {
  const [count, setCount] = React.useState(0);
  console.log("app render", count);

  return (
    <div>
      <button onClick={() => setCount(c => c + 1)}>{count}</button>
      <MemoChild label="stable" />
    </div>
  );
}

render(<App />);
```

## 参考答案
首次渲染会打印 `app render 0`、`child render stable`。点击一次后只会打印 `app render 1`，子组件不会重新渲染。

## 考察点
- React.memo 只做浅比较
- 稳定的基础类型 props 可以让 React 跳过子组件重渲染

## 详细讲解
这里传给子组件的是稳定的字符串 props。`React.memo` 会对 props 做浅比较，发现 `label` 没有变化，因此父组件重新渲染后，子组件会被跳过，不会再次执行。
