# React 输出题 35：useReducer 的 init 函数只在挂载时执行一次

## 基本信息
- 分类：Hooks / useReducer
- 难度：初级
- 标签：useReducer, 初始化函数, 挂载
- 题型：阅读 React 代码并写出输出

## 题目描述
点击之后 `init` 函数还会不会再次执行？

## 代码
```jsx
function init(value) {
  console.log("init");
  return value * 2;
}

function reducer(state, action) {
  if (action.type === "add") return state + 1;
  return state;
}

function App() {
  const [count, dispatch] = React.useReducer(reducer, 1, init);
  console.log("render", count);

  return <button onClick={() => dispatch({ type: "add" })}>{count}</button>;
}

render(<App />);
```

## 参考答案
输出顺序是：`init -> render 2`，点击一次后再打印 `render 3`。

## 考察点
- init 只会在 reducer 初始状态建立时执行一次

## 详细讲解
`useReducer` 的第三个参数 `init` 只会在 hook 首次挂载时用于计算初始值。后续点击触发的是 reducer 计算，不会重新执行 `init`，所以只会看到 `render 3`。
