# React Output Question 35: useReducer init function runs once on mount

## Basic Info
- Category: Hooks / useReducer
- Difficulty: Junior
- Tags: useReducer, init function, mount
- Type: Read React code and write the output

## Description
点击之后 `init` 函数还会不会再次执行？

## Code
```jsx
function init(value) {
  console.log("init");
  return value * 2;
}

function reducer(state, action) {
  if (action.type === "add") return state + 1;
  return state;
}

function App() {
  const [count, dispatch] = React.useReducer(reducer, 1, init);
  console.log("render", count);

  return <button onClick={() => dispatch({ type: "add" })}>{count}</button>;
}

render(<App />);
```

## Answer
Output: init, render 2, then after one click render 3.

## Key Points
- init runs only for the initial reducer state

## Detailed Explanation
`useReducer` 的第三个参数 `init` 只会在 hook 首次挂载时用于计算初始值。后续点击触发的是 reducer 计算，不会重新执行 `init`，所以只会看到 `render 3`。
