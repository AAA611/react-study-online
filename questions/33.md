# React 输出题 33：interval 里的函数式更新能拿到最新 state

## 基本信息
- 分类：Hooks / 函数式更新
- 难度：中级
- 标签：函数式更新, 定时器, 最新状态
- 题型：阅读 React 代码并写出输出

## 题目描述
这次每秒钟会打印什么？

## 代码
```jsx
function App() {
  const [count, setCount] = React.useState(0);

  React.useEffect(() => {
    const id = setInterval(() => {
      setCount(c => {
        console.log(c);
        return c + 1;
      });
    }, 1000);

    return () => clearInterval(id);
  }, []);

  return null;
}

render(<App />);
```

## 参考答案
它会持续打印 `0`、`1`、`2`、`3`……不断递增。

## 考察点
- 函数式 updater 总能拿到最新状态

## 详细讲解
这里打印发生在函数式更新的回调内部，参数 `c` 是 React 传进来的最新 state，而不是旧闭包中的值。因此它会依次打印 `0、1、2、3`，不会卡在旧值上。
