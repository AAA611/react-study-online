# React Output Question 33: Functional updates inside interval can see the latest state

## Basic Info
- Category: Hooks / functional update
- Difficulty: Intermediate
- Tags: functional update, interval, latest state
- Type: Read React code and write the output

## Description
这次每秒钟会打印什么？

## Code
```jsx
function App() {
  const [count, setCount] = React.useState(0);

  React.useEffect(() => {
    const id = setInterval(() => {
      setCount(c => {
        console.log(c);
        return c + 1;
      });
    }, 1000);

    return () => clearInterval(id);
  }, []);

  return null;
}

render(<App />);
```

## Answer
It prints 0, 1, 2, 3, and so on.

## Key Points
- functional updater receives the freshest state

## Detailed Explanation
这里打印发生在函数式更新的回调内部，参数 `c` 是 React 传进来的最新 state，而不是旧闭包中的值。因此它会依次打印 `0、1、2、3`，不会卡在旧值上。
