# React Output Question 19: Objects created inside a component are new every render

## Basic Info
- Category: Render model / references
- Difficulty: Junior
- Tags: object reference, rerender, identity
- Type: Read React code and write the output

## Description
点击一次后，`same?` 比较结果是什么？

## Code
```jsx
function App() {
  const [count, setCount] = React.useState(0);
  const obj = { value: 1 };
  const prevRef = React.useRef();

  console.log("same?", prevRef.current === obj);
  prevRef.current = obj;

  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}

render(<App />);
```

## Answer
First render prints same? false. After one click it prints same? false again.

## Key Points
- component body runs again on every render
- object literals are recreated each time

## Detailed Explanation
函数组件每次重新渲染都会重新执行整个函数体，因此对象字面量 `obj` 也会被重新创建。`prevRef.current` 存的是上一次渲染中的对象，所以无论首次还是更新后，比较结果都会是 `false`。
