# React Output Question 45: Multiple useEffect hooks run in declaration order

## Basic Info
- Category: Hooks / ordering
- Difficulty: Junior
- Tags: useEffect, order, multiple effects
- Type: Read React code and write the output

## Description
同一个组件里声明两个 effect，首次挂载时它们的执行顺序是什么？

## Code
```jsx
function App() {
  console.log("render");

  React.useEffect(() => {
    console.log("effect 1");
  }, []);

  React.useEffect(() => {
    console.log("effect 2");
  }, []);

  return null;
}

render(<App />);
```

## Answer
Output: render -> effect 1 -> effect 2.

## Key Points
- effects are executed in declaration order

## Detailed Explanation
组件函数先执行，所以 `render` 最先出现。提交完成后，React 会按照代码中声明的顺序依次执行 effect，因此 `effect 1` 先于 `effect 2`。
