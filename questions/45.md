# React 输出题 45：多个 useEffect 会按声明顺序执行

## 基本信息
- 分类：Hooks / 执行顺序
- 难度：初级
- 标签：useEffect, 顺序, 多个 effect
- 题型：阅读 React 代码并写出输出

## 题目描述
同一个组件里声明两个 effect，首次挂载时它们的执行顺序是什么？

## 代码
```jsx
function App() {
  console.log("render");

  React.useEffect(() => {
    console.log("effect 1");
  }, []);

  React.useEffect(() => {
    console.log("effect 2");
  }, []);

  return null;
}

render(<App />);
```

## 参考答案
输出顺序是：`render -> effect 1 -> effect 2`。

## 考察点
- 多个 effect 会按照声明顺序执行

## 详细讲解
组件函数先执行，所以 `render` 最先出现。提交完成后，React 会按照代码中声明的顺序依次执行 effect，因此 `effect 1` 先于 `effect 2`。
