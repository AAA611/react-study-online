# React 输出题 38：0 && JSX 的结果是 0，不是什么都不渲染

## 基本信息
- 分类：JSX / 基础
- 难度：初级
- 标签：JSX, 逻辑与, 渲染
- 题型：阅读 React 代码并写出输出

## 题目描述
页面上最终会显示什么文本？

## 代码
```jsx
function App() {
  return <div>{0 && <span>React</span>}</div>;
}

render(<App />);
```

## 参考答案
页面上会直接显示 `0`。

## 考察点
- `0` 在 React 里是一个可以被渲染出来的真实值
- 逻辑与表达式在左侧为假值时会直接返回左侧

## 详细讲解
JavaScript 的 `&&` 运算在左侧是假值时会直接返回左侧本身，这里返回的是数字 `0`。React 会忽略 `false`、`null`、`undefined`，但不会忽略数字 `0`，所以页面上会真正显示 `0`。
