# React Output Question 38: 0 && JSX returns 0, not nothing

## Basic Info
- Category: JSX basics
- Difficulty: Junior
- Tags: JSX, logical and, rendering
- Type: Read React code and write the output

## Description
页面上最终会显示什么文本？

## Code
```jsx
function App() {
  return <div>{0 && <span>React</span>}</div>;
}

render(<App />);
```

## Answer
The page shows 0.

## Key Points
- 0 is a real renderable value in React
- logical and returns the left side when it is falsy

## Detailed Explanation
JavaScript 的 `&&` 运算在左侧是假值时会直接返回左侧本身，这里返回的是数字 `0`。React 会忽略 `false`、`null`、`undefined`，但不会忽略数字 `0`，所以页面上会真正显示 `0`。
