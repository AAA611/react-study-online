# React 输出题 43：value 会让 input 成为受控组件

## 基本信息
- 分类：表单 / 受控输入
- 难度：初级
- 标签：value, 受控输入, 表单
- 题型：阅读 React 代码并写出输出

## 题目描述
如果把上一题改成 `value`，最终 input 会显示什么？

## 代码
```jsx
function App() {
  const [text, setText] = React.useState("hello");

  React.useEffect(() => {
    setText("world");
  }, []);

  return <input value={text} readOnly />;
}

render(<App />);
```

## 参考答案
输入框里会显示 `world`。

## 考察点
- 受控输入框的 `value` 会始终跟随 state

## 详细讲解
这里的 input 是受控组件，显示内容完全由 React state 决定。state 更新为 `world` 后，组件重新渲染，input 的 `value` 也会同步更新成 `world`。
