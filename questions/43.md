# React Output Question 43: value makes the input controlled

## Basic Info
- Category: Forms / controlled input
- Difficulty: Junior
- Tags: value, controlled input, forms
- Type: Read React code and write the output

## Description
如果把上一题改成 `value`，最终 input 会显示什么？

## Code
```jsx
function App() {
  const [text, setText] = React.useState("hello");

  React.useEffect(() => {
    setText("world");
  }, []);

  return <input value={text} readOnly />;
}

render(<App />);
```

## Answer
The input shows world.

## Key Points
- controlled input value always follows state

## Detailed Explanation
这里的 input 是受控组件，显示内容完全由 React state 决定。state 更新为 `world` 后，组件重新渲染，input 的 `value` 也会同步更新成 `world`。
